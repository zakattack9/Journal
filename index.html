Difference Between UX and UI

UX Stands for User Experience, the users experience while using a website or application; the user's experience may be pleased if the website is organized and interactive with the user.

UI Stands for User Interface, the graphical desgin of an application or website; this is what the users sees while using the website, it is often colorful for better eye appeal.

HTML & CSS

September 7, 2016

HTML

HTML or HyperText Markup Language, is a computer language used to describe a web page; because it is a markup language, it includes many tags that add elements to a page such as paragraphs and lists. HTML tells the web browser how everything is to be organized and like any other language includes numerous syntax. Without HTML the web page has no structure and therefore cannot exist.

CSS

CSS or Cascading Style Sheets, manages the design and presentation of a web page. CSS controls the visual appeal to a viewer as it can change the color, border, font and much more. It is often the UI of a website and kept on a seperate file from the HTML. Think of HTML and CSS as a cake, HTML being the cake itself, the founation and stucture of the web page and CSS as the icing on the cake, what makes it visually appealing and colorful.
